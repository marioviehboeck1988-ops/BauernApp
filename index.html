Firebase-Einrichtung

1

Firebase-Konto erstellen

Gehen Sie auf firebase.google.com und melden Sie sich mit einem Google-Konto an.

2

Neues Projekt anlegen

Klicken Sie auf „Projekt erstellen", geben Sie einen Namen ein (z.B. efko-wochenmeldung) und deaktivieren Sie Google Analytics.

3

Web-App registrieren

Klicken Sie im Projekt auf das </>-Symbol, geben Sie einen App-Namen ein und klicken Sie auf „App registrieren".

4

Realtime Database anlegen

Wählen Sie links „Realtime Database" → „Datenbank erstellen" → Standort europe-west1 → „Im Testmodus starten". Dann unter „Regeln":

{"rules": {".read": true, ".write": true}}
5

Konfiguration eintragen

Öffnen Sie diese HTML-Datei im Texteditor und suchen Sie nach FIREBASE_CONFIG_HERE. Ersetzen Sie den gesamten Abschnitt mit Ihrer Firebase-Konfiguration.

6

Datei auf GitHub Pages ablegen

Die fertig konfigurierte HTML-Datei auf GitHub Pages hochladen und die URL per WhatsApp an alle Produzenten schicken.

Verbindung wird hergestellt…
